Please install the correct packages before starting this project:
	-python-3.6.4
	-chromedriver.exe and geckodriver.exe need to be moved to a folder "C:/drivers" (if already taken find another name)
	    -Adding both to path

	Python Package Installs (in directory where project is):
		Selenium - pip install selenium
		Requests - pip install requests
		BeautifulSoup4 - pip install beautifulsoup4

	python bot.py starts the bot when in the directory, type help for more
